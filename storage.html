What is Local Storage?

  • window.localStorage is a property that allows users to store data values in a web browser.
  • Usually, stored datas are deleted when browser is closed. However, localStorage stores data for an unlimited time.

Example

Step 1: Create an Object

Car = {make: "Nissan", model: "Sentra", year: 2015}

Step 2: Stringify Object

thisCar = JSON.stringify(Car)
localStorage. setItem("mycar", thisCar)

Step 3: Get the String

textCar = localStorage. getItem("mycar")
newCar = JSON.parse(textCar)
console.log(newCar)
